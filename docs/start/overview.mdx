# Ante

**An**other **Te**rminal — a lightweight AI agent that lives in your terminal, built by [Antigma Labs](https://antigma.ai).
A precursor to our mission of "building substrate for self-organizing intelligence."

Ante is built from the ground up in native Rust for security, performance, and resistance to AI-generated slop — a single self-contained binary with no external dependencies.

### Key features

<CardGroup cols={3}>
  <Card title="Lightweight agent core" icon="feather">
    A single ~15MB binary with zero runtime dependencies. The agent core is built for minimal overhead and maximum throughput — making it the ideal runtime for orchestrating agent swarms at scale.
  </Card>
  <Card title="Native local models" icon="microchip">
    Run models entirely on your machine with built-in llama.cpp integration. No API keys, no internet, no data leaving your device. Ante handles engine installation, model discovery, and memory management automatically.
  </Card>
  <Card title="Zero vendor lock-in" icon="lock-open">
    Bring your own API key, subscription, or local model. Switch between 12+ providers freely — Anthropic, OpenAI, Gemini, Grok, Open Router, and more. No account required, no telemetry gate, no walled garden. Not even ours.
  </Card>
</CardGroup>

### Why Ante?

- **Obsessed with simplicity and quality.** In a world of abundance, maintaining quality and a high level of trust is more valuable than ever.
- **Self-contained with [offline mode](/offline).** We strive to close the loop between training and inference.
- **[Meta Agent Orchestration](/agent-org).** Coordinate multiple agents that self-organize around complex tasks.
- **Benchmark proven.** See [Benchmarks](/benchmarks/eval).

### How it works

Ante runs as an interactive agent in your terminal. You give it a task in natural language, it reasons through a plan, executes shell commands and file edits with your approval, and iterates until the job is done. Under the hood, Ante manages a structured turn lifecycle — prompt, tool calls, confirmation, execution — so the model's power is channeled through a safe, predictable loop.

Ante can also run as a long-lived server (`ante serve`) that external clients drive through a structured JSONL protocol — ideal for building editor plugins, web UIs, and custom integrations.

<Warning>
  Ante is currently in **preview** and under active development. Expect breaking changes, experimental features, and incomplete functionality. Currently only **macOS** and **Linux** are supported.
</Warning>
